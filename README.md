# BlackServ SSH — aktualizacje

Publiczne repozytorium dystrybucyjne aplikacji **BlackServ SSH**.

Repozytorium zawiera wyłącznie publiczne pliki aktualizacji:

- `latest.json` — manifest najnowszej wersji,
- podpisane pakiety APK publikowane w sekcji **Releases**.

Kod źródłowy aplikacji, klucz podpisujący i dane dostępowe nie są tu przechowywane.

## Bezpieczeństwo

Przed uruchomieniem instalatora aplikacja weryfikuje sumę SHA-256 pobranego APK. Wszystkie wersje muszą być podpisane tym samym kluczem co poprzednie wydania.
